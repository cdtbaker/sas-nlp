To install SCANComment as a user in eclipse go to help > install new software and select 'Add' then navigate the local update site (./ScanCommentUDSite). Follow the wizard through and SCANComment with all of its current analyses will be installed. Before installing SCANComment make sure that the version of Crystal found in /src/crystal is installed from the update site provided.

To run a SCANComment analysis on your code in the Crystal menu on the toolbar make sure that ScanCommentAnalysis is ticked.
